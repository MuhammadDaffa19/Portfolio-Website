PORTFOLIO WEBSITE REQUIREMENTS
Project Name: Daffa Project Index
Project Type: Static Interactive Portfolio Website
Design Concept: Space-themed Interactive Portfolio Gateway / Orbital Navigation
Language: Indonesian default, English translation available

==================================================
1. CORE FRONTEND TECHNOLOGY
==================================================
- React
  Purpose:
  Main frontend library for building reusable UI components.
  Used for:
  - Homepage interactive gateway
  - Orbital navigation component
  - Project archive pages
  - About, skills, experience, certificate, and contact pages
  - Language toggle ID/EN

- Vite
  Purpose:
  Fast development server and build tool for React.
  Used for:
  - Local development
  - Production build
  - Static deployment output

- JavaScript / JSX
  Purpose:
  Main programming language for UI logic and component behavior.
  Used for:
  - State management for menu open/close
  - Hover and click interaction
  - Language switching
  - Project data rendering

==================================================
2. STYLING TECHNOLOGY
==================================================
- Tailwind CSS
  Purpose:
  Main utility-first CSS framework for styling.
  Used for:
  - Layout
  - Spacing
  - Typography
  - Responsive design
  - Glassmorphism panels
  - Button states
  - Dark cosmic theme styling

- Custom CSS
  Purpose:
  Additional styling for unique visual effects not ideal to write fully in Tailwind.
  Used for:
  - Orbital rings
  - Glow effects
  - Background overlay
  - Keyframe animations
  - Custom hover effects
  - Fine-tuned responsive adjustments

==================================================
3. ANIMATION TECHNOLOGY
==================================================
- Motion for React / Framer Motion
  Purpose:
  Main React animation library for UI transitions.
  Used for:
  - Initial page reveal
  - Text fade-in
  - Orbital core entrance animation
  - Button expand animation
  - Hover scale animation
  - Page transitions
  - AnimatePresence transitions between routes/components

- GSAP
  Purpose:
  Advanced animation library for more complex motion.
  Used for:
  - Orbital ring animation
  - Navigation node stagger animation
  - Core activation glow
  - Portal-like click transition
  - Smooth timeline-based animation sequences

==================================================
4. ROUTING AND PAGE STRUCTURE
==================================================
- React Router
  Purpose:
  Client-side routing for multiple static portfolio pages.
  Planned routes:
  - /                 : Home / Interactive Gateway
  - /about            : Tentang Saya
  - /projects         : Project Archive
  - /projects/:slug   : Project Detail / Case Study
  - /skills           : Keahlian
  - /experience       : Pengalaman
  - /certificates     : Sertifikat
  - /contact          : Kontak

==================================================

  Keep the website bilingual without using a heavy translation library.
  Default language:
  - Bahasa Indonesia
  Secondary language:
  - English

Planned feature:
- ID / EN toggle button
- Text content changes based on selected language
- Language preference can optionally be saved in localStorage

==================================================
8. MEDIA AND IMAGE REQUIREMENTS
==================================================
- WebP / AVIF Images
  Purpose:
  Optimized image format for performance.
  Used for:
  - Space background image
  - Project screenshots
  - Mockup previews
  - Certificate images

- Image Optimization Rules
  Requirements:
  - Compress all large images before production
  - Use WebP as default where possible
  - Avoid oversized PNG/JPG files
  - Use lazy loading for project images below the first screen

==================================================
9. DESIGN REQUIREMENTS
==================================================
- Homepage style:
  Space-themed interactive gateway inspired by the interaction concept of ASN Digital, but visually original.

- Main visual direction:
  Dark cosmic background, orbital navigation, premium glassmorphism, glowing interface accents.

- Homepage states:
  1. Initial state:
     - Headline visible
     - Orbital core visible
     - Quote visible
     - Navigation buttons hidden

  2. Active state:
     - Orbital core glows brighter
     - Orbit lines activate
     - Six navigation buttons appear around the core

- Main navigation buttons:
  - Tentang Saya
  - Proyek
  - Keahlian
  - Pengalaman
  - Sertifikat
  - Kontak

- Avoid:
  - Generic AI gradient style
  - Excessive neon effects
  - Copying ASN Digital colors, map, icons, or layout exactly
  - Unnecessary buttons
  - Heavy 3D effects
  - Overcrowded particles

==================================================
10. RESPONSIVE REQUIREMENTS
==================================================
- Desktop:
  - Full interactive orbital gateway
  - Hover-based menu expansion
  - Expanded navigation nodes around central core

- Tablet:
  - Simplified orbital layout
  - Touch-friendly buttons
  - Adjusted spacing

- Mobile:
  - Tap core to open navigation
  - Navigation buttons displayed in a clean stacked or radial simplified layout
  - Avoid hover-only interactions
  - Maintain readability and performance

==================================================
11. PERFORMANCE REQUIREMENTS
==================================================
- Website must remain static and lightweight.
- Avoid unnecessary backend.
- Avoid heavy 3D/WebGL unless truly needed.
- Optimize all images.
- Use lazy loading for lower-priority assets.
- Keep animation smooth but not excessive.
- Ensure the first screen loads quickly.

Recommended approach:
- Use background image instead of real-time particle canvas.
- Use CSS/SVG/Motion/GSAP for interactive orbital effects.
- Do not use Three.js for the first version.

14. PROJECT INSTALLATION PLAN
==================================================
Initial setup commands:

npm create vite@latest daffa-project-index -- --template react
cd daffa-project-index
npm install

Install main dependencies:

npm install tailwindcss @tailwindcss/vite
npm install motion
npm install gsap
npm install react-router-dom
npm install lucide-react

Run development server:

npm run dev

Build production files:

npm run build
